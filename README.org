* OCaml version of Category Theory For Programmers

** How do I get the book?
   - Currently, the book can be download by cloning this fork -
     https://github.com/ArulselvanMadhavan/milewski-ctfp-pdf and
     running
     #+BEGIN_SRC bash
     cd src
     make ocaml
     #+END_SRC
